해당 프로젝트는 2025년 6월 24일 부터 9월 11일 까지 Unity Version Control을 통해 버전관리를 진행한 프로젝트를 공개 목적으로  
깃허브에 업로드 하였습니다. [Unity Version Control 기록](https://github.com/woong020477/TerionTD/tree/main/Info_Data/history)  
  
공백 기간동안 개선되지 않은 문제점을 수정하기 위해 시간을 내려고 노력하고 있으나 쉽지 않은 상황입니다.  
따라서 인지는 하고있지만 빠른 시일내에 해결되지 않을 수 있습니다.

* 게임 패배 시 방이 제거되지 않고 서버에 남아있는 현상
